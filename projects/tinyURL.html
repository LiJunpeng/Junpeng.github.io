URL Shortener Service

Tweet your long URL

Key words: Node.js, Express, AngularJS, MongoDB, Docker, Redis

Twitter has a character limit on each tweet, which gives me a hard time to share long urls with friends.

The URL Shortener Service is the solution! This project is similar to Google URL Shortener or other shortener service. Basically, it takes a long url and return a short one back to user. User can then uses the shorter one to access the original web page that long url points to.

I also add statistic functionality that allows user to track how many people click the short url he created, which browser people were using when clicked the short url, etc.

The project is designed to be deployed in a dockerized fashion which integrated Redis as cache and Nginx as load balancer, or as a standalone Node.js program. Here is the github link to the project: https://github.com/LiJunpeng/URLShortenerService
I also have a demo deployed on AWS. Try it out!
