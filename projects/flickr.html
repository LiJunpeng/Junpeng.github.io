Flickr Gallery

Enjoy some random photos from Flickr, and relax!

Key words: AngularJS, Bootstrap, NodeJS, Express

"Mini-flickr" is always a good project idea for front-end practice, this is my version of it!

I have developed a Node.js server program to deal with the Flickr API and handle topic search. For each search request, the server fetches 100 urls of photos from Flickr and pass them to the front-end. Once received results, front-end loads photos in a 'lazy-load' fashion. There is also a 'filter' for user to search photos with a specific tag.

When browsing photos, user can always click the tiny star at the right bottom corner of the photo card to 'pin' it. Only 'pinned' photos will be displayed when user switch from 'All' tab to 'Pinned' tab.

Here is the github link to the project: https://github.com/LiJunpeng/flickr_gallery
I have a demo deployed on AWS. Try it out!
