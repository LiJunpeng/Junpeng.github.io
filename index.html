URL Shortener Service

This project is similar to Google URL Shortener or other shortener service. Basically, it takes a long url and return a short one back to user. User can then uses the shorter one to access the original web page that long url points to. It's a useful service when posting url in social network like Twitter who has length limit. I also add statistic functionality that allows user to track how many people click the short url he created.

Key words: Node.js, Express, AngularJS, MongoDB, Docker, Redis
